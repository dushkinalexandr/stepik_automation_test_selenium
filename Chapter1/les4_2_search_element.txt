CSS selector
###################
https://suninjuly.github.io/cats.html
===================
id (завист от регистра)
[id="bullet"] or #bullet

===================
tag (тэг без символов; не удобно, т.к. тэгов одинаковых много)
h1, a, div

===================
значение атрибута
[value="Cat memes"]

===================
name (частый способ поиска)
[name="bullet-cat"]

===================
class (зависит от регистра и порядка классов)
[class="lead text-muted"] - ищет точное совпадение
.lead - ищет вхождение

поиск по нескольким классам
[class="lead text-muted"] - важен порядок классов и регистр
.text-muted.lead or .lead.text-muted - не важен порядок, важен регистр
соседи не разделяются пробелом


###################
составной CSS selector
###################
http://suninjuly.github.io/blog_example.html

===================
Использование потомков
#post2 .title - #id(предок) .class(потомок) - потомок отделяется проблелом
#posts .title - потомок может нахоидиться на любом уровне вложенности - 3 совпадения найдёт

===================
Использование дочерних элементов
#post2 > div.title
> - указывает строго на один уровень ниже, можно не отделять пробелами
#post2 - parent
div.title - child

#post2 div.title - будет выполнен поиск всех элементов "div.title" на любом уровне ниже первого элемента

===================
Использование порядкового номера дочернего элемнета
"#posts > .item:nth-child(2) > .title"

Псевдо-класс :nth-child(2) — позволяет найти второй по порядку элемент среди дочерних элементов для "#posts".
Затем с помощью "> .title" мы указываем, что нам нужен элемент ".title", родителем которого является найденный ранее элемент ".item".

===================
Использование нескольких классов
".title.second"
Также мы можем использовать сразу несколько классов элемента, чтобы его найти.
Для этого классы записываются подряд через точку: ".title.second"
